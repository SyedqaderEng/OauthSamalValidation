# MockAuth Complete Demo Pages

This is a complete interactive HTML demo of the MockAuth platform.
You can click through all pages to experience the full application flow.

## Pages Included:

### Authentication
- login.html - Login page
- signup.html - Signup page  
- forgot-password.html - Password recovery

### Dashboard
- dashboard.html - Main dashboard
- index.html - Landing page

### OAuth
- oauth-create.html - Create OAuth app (pre-filled form)
- oauth-detail.html - View OAuth app details
- oauth-list.html - List all OAuth apps

### SAML
- saml-sp-create.html - Create SAML SP (pre-filled form)
- saml-idp-create.html - Create SAML IdP (pre-filled form)
- saml-detail.html - View SAML environment details
- saml-list.html - List all SAML environments

### Tools
- jwt-decoder.html - JWT decoder tool
- saml-decoder.html - SAML decoder tool
- base64-tools.html - Base64 encoder/decoder

### Settings
- settings-profile.html - User profile settings
- settings-team.html - Team settings
- settings-api-keys.html - API key management
- settings-billing.html - Billing settings

### Other
- docs.html - Documentation page
- support.html - Support page

## How to Use

1. Start with index.html (landing page)
2. Click "Get Started" to go to signup
3. Or click login to see the login page
4. Navigate through the dashboard and all features
5. All forms have pre-filled sample data to show what users would enter

All pages are fully linked and styled consistently.
