4. saml_config          - Application SAML configuration

API ENDPOINTS:
==============
Authentication:
  POST   /api/auth/signup
  POST   /api/auth/login
  GET    /api/auth/me
  GET    /api/auth/saml-logs

Metadata:
  POST   /api/metadata/import
  GET    /api/metadata/export/sp
  GET    /api/metadata/export/idp
  GET    /api/metadata/list
  GET    /api/metadata/:entityId
  DELETE /api/metadata/:entityId

SAML Configuration:
  GET    /api/saml-config
  PUT    /api/saml-config

SAML SP Endpoints:
  GET    /saml/metadata
  GET    /saml/login/:idpEntityId
  POST   /saml/acs
  GET    /saml/slo

SAML IdP Endpoints:
  GET    /saml/idp/metadata
  GET    /saml/idp/sso
  POST   /saml/idp/sso/process
  GET    /saml/idp/slo

Logs:
  GET    /saml/logs
